Win10-cmd-PowerShell-SSHж— жі•еҲӣе»әзӣ®еҪ•-е’Ң-SCP-жүҫдёҚеҲ°ж–Үд»¶

еҸ‘иЎЁдәҺ 2020-02-17 жӣҙж–°дәҺ 2020-08-21 еҲҶзұ»дәҺ е·Ҙе…·
жң¬ж–Үеӯ—ж•°пјҡ 956 йҳ…иҜ»ж—¶й•ҝ ≈ 2 еҲҶй’ҹ

й—®йўҳ: SSHж— жі•жүҫеҲ°зӣ®еҪ•е’Ңж–Үд»¶

й”ҷиҜҜдҝЎжҒҜ

иҫ“е…ҘSSH root@remotenode

иҫ“еҮә:
ж— жі•еҲӣе»әзӣ®еҪ• Could not create directory: C:\users\123\456\789.ssh
ж— жі•ж·»еҠ дҝЎд»»дё»жңәеҲ—иЎЁ Failed to addвҖҰ..

з„¶иҖҢ.sshзӣ®еҪ•жҳҺжҳҺе°ұеңЁз”ЁжҲ·еҗҚж–Үд»¶еӨ№дёӢеӯҳеңЁ= =

иҫ“е…Ҙscp ./test.txt root@remotenode:/root/test.win

ж— жі•жүҫеҲ°ж–Үд»¶ : No Such File
дәӢе®һдёҠиҝҷдёӘж–Үд»¶еӯҳеңЁ

еҺҹеӣ зҢңжөӢ: з”ЁжҲ·еҗҚеёҰдёӯж–ҮжҲ–зү№ж®Ҡеӯ—з¬Ұ, еҜјиҮҙи·Ҝеҫ„ж— жі•иҜҶеҲ«

cmdе’ҢPowerShell зҡ„зј–з ҒйғҪжҳҜGBK, и®ІйҒ“зҗҶжҳҜеҸҜд»Ҙз”Ёдёӯж–Үзҡ„вҖҰ

дҪҶжҳҜдҪҝз”ЁSSHе‘Ҫд»ӨжҳҫиҖҢжҳ“и§ҒеҸҜд»ҘзңӢеҲ°дёӯж–ҮеҗҚиў«и§ЈжһҗжҲҗдәҶиҪ¬д№үеӯ—з¬Ұ вҖң\323вҖқ д№Ӣзұ»зҡ„.

дҪҝз”ЁchжүҫеҲ°зӣ®еҪ•е’Ңж–Үд»¶

SSH root@remotenode

иҫ“еҮә:

ж— жі•еҲӣе»әзӣ®еҪ• Could not create directory: C:\users\123\456\789.ssh
ж— жі•ж·»еҠ дҝЎд»»дё»жңәеҲ—иЎЁ Failed to addвҖҰ..

й…ҚзҪ®зҡ„е…ҚеҜҶзҷ»еҪ•иҝңзЁӢдё»жңәд№ҹдёҚиғҪз”Ё

з„¶иҖҢ.sshзӣ®еҪ•жҳҺжҳҺе°ұеңЁз”ЁжҲ·еҗҚж–Үд»¶еӨ№дёӢеӯҳеңЁ= =

scp ./test.txt root@remotenode:/root/test.win

ж— жі•жүҫеҲ°ж–Үд»¶ : No Such File
дәӢе®һдёҠиҝҷдёӘж–Үд»¶еӯҳеңЁ

еҺҹеӣ зҢңжөӢ: з”ЁжҲ·еҗҚеёҰдёӯж–ҮжҲ–зү№ж®Ҡеӯ—з¬Ұ, еҜјиҮҙи·Ҝеҫ„ж— жі•иҜҶеҲ«

cmdе’ҢPowerShell зҡ„зј–з ҒйғҪжҳҜGBK, и®ІйҒ“зҗҶжҳҜеҸҜд»Ҙз”Ёдёӯж–Үзҡ„вҖҰ

дҪҶжҳҜдҪҝз”ЁSSHе‘Ҫд»ӨжҳҫиҖҢжҳ“и§ҒеҸҜд»ҘзңӢеҲ°дёӯж–ҮеҗҚиў«и§ЈжһҗжҲҗдәҶиҪ¬д№үеӯ—з¬Ұ вҖң\323вҖқ д№Ӣзұ»зҡ„.

дҪҝз”Ёchcp 65001жҚўжҲҗutf-8д№ҹдёҚиЎҢ

еҪ“з„¶д№ҹдёҚеҸҜиғҪйҮҚе‘ҪеҗҚз”ЁжҲ·еҗҚж–Үд»¶еӨ№..дёҖеӨ§е ҶзЁӢеәҸж•°жҚ®йғҪеңЁдёӢйқўе‘ў= =

жҲ‘зҡ„еҸҜиЎҢж–№жЎҲ :

дҪҝз”Ёgit bash д»Јжӣҝ[cmd/PowerShel]

д№ӢеүҚиЈ…gitжңүиЈ…git bash. е№¶дё”git bash йҮҢеҸҜд»Ҙи®ҫзҪ®дёӯж–Үеӯ—з¬ҰйӣҶ,еӣ жӯӨе°ұиҜ•дәҶдёҖдёӢ, еҸ‘зҺ°е®Ңе…ЁжІЎжңүйҡңзўҚ= =

зҺҜеўғд»Җд№Ҳзҡ„е®Ңе…ЁжІЎеҸҳ, еҸҜд»ҘжӯЈеёёжүҫеҲ°.sshзӣ®еҪ• , еҸҜд»ҘжӯЈеёёеҶҷе…Ҙ known_host ,йҷӨдәҶеңЁз¬¬дёҖж¬Ўзҷ»зҡ„ж—¶еҖҷиҜўй—®еҶҷе…Ҙknown_host, д№ӢеҗҺе…ҚеҜҶзҷ»еҪ•д№ҹеҸҜд»ҘжӯЈеёёдҪҝз”Ё.

scpеҸҜд»ҘжӯЈеёёеҸ‘йҖҒз”ЁжҲ·еҗҚзӣ®еҪ•дёӢзҡ„ж–Үд»¶, дёҚдјҡжүҫдёҚеҲ°.

жһңз„¶жҹҘйҒҚе…ЁзҪ‘д№ҹи§ЈеҶідёҚдәҶзҡ„й—®йўҳжҳҜcmdиҮӘиә«зҡ„й—®йўҳ= =

дҪҝз”ЁSSHе·Ҙе…·,еҰӮPutty

дҪҝз”ЁPutty openзҷ»еҪ•дё»жңә, еҘҪеғҸдёҚдјҡжҸҗзӨәknown_host зҡ„дәӢжғ…, дҪҶжҳҜеҸҜд»ҘйЎәеҲ©е…ҚеҜҶзҷ»еҪ•.

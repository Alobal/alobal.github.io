队列

发表于 2020-08-06 分类于 数据结构

概述

FIFO的数据结构，先进入的元素先出来。

队列操作只能在两端进行。尾部入队，头部出队。

可以使用 数组，链表 进行实现。

vector实现

利用vector往后存储，但是出队后会浪费前面的空间

class MyQueue {
    private:
        // 元素存储列表
        vector<int> data;      //需要<vector> 
        // 头部指针，指向最早的元素
        int p_start;            
    public:
        MyQueue() {p_start = 0;}
        /** 入队,注意p_start仍指向最早的元素 */
        bool enQueue(int x) {
            data.push_back(x);
            return true;
        }
        /** 出队,p_start后移 */
        bool deQueue() {
            if (isEmpty()) {
                return false;
            }
            p_start++;
            return true;
        };
        /** 获取最前元素 */
        int Front() {
            return data[p_start];
        };
        /**  */
        bool isEmpty()  {
            return p_start >= data.size();
        }
};
